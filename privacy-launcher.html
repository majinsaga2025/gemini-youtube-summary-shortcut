Privacy Policy

Last Updated: January 12, 2025

This privacy policy applies to the following Android applications developed by majinsaga2025:

  • Private Launcher - A dedicated launcher for Android 15's Private Space feature
  • HomeJump - A utility app to quickly access home app (launcher) settings

These applications respect user privacy and do not collect or store personal information.

Data Collection

These applications do not collect, track, or store any personal data.

We do not use analytics, advertising SDKs, or any third-party tracking services.

Local Storage

Both applications store user settings locally on the device using Android's SharedPreferences. These settings are stored only on the user's device and are never transmitted to any external servers.

Private Launcher

Data Usage

This application only accesses the list of apps installed in your Private Space to display them on screen. This information is:

  • Processed entirely on your device
  • Never transmitted to any external servers
  • Not stored persistently (retrieved fresh each time the app is opened)

Local Storage

  • Theme settings: Light, Dark, or System theme preference
  • Language settings: Preferred app language (Japanese, English, Spanish, or System)
  • Auto-unlock settings: Whether to automatically request unlock on app launch
  • Favorite apps: List of selected apps to display and their order

Permissions

This application requires the following permissions to function as a Private Space launcher:

  • ACCESS_HIDDEN_PROFILES: Required to detect and access the Private Space profile. This is an Android 15+ API specifically designed for apps that need to interact with Private Space.
  • QUERY_ALL_PACKAGES: Required to retrieve the list of apps installed in Private Space. This is a standard permission for launcher applications.

These permissions are essential for the core functionality of this launcher app. The information obtained through these permissions is used solely to display your Private Space apps and is never transmitted externally.

Default Launcher Requirement

To access Private Space, this app must be set as the default home launcher. This is a requirement imposed by Android's Private Space security model, not a choice by the developer. You can switch back to your preferred launcher at any time through Android Settings.

HomeJump

Data Usage

This application does not access any user data. It simply provides a shortcut to open Android's home app settings screen.

Local Storage

  • Skip main screen setting: Whether to skip the main screen and go directly to launcher settings
  • Auto-transition seconds: Delay before automatically opening launcher settings
  • Theme settings: Light, Dark, or System theme preference
  • Language settings: Preferred app language (Japanese, English, Spanish, or System)

Permissions

This application requires no special permissions.

The only system permission used is BIND_QUICK_SETTINGS_TILE, which is a standard Android permission for apps that provide Quick Settings tiles. This permission does not access any user data.

Children's Privacy

These applications are not directed at children under 13 years of age. We do not knowingly collect personal information from children.

Changes to This Policy

This privacy policy may be updated if the applications' functionality changes. Any updates will be reflected on this page with a new "Last Updated" date.

プライバシーポリシー

最終更新日: 2025年1月12日

本プライバシーポリシーは、majinsaga2025が開発した以下のAndroidアプリケーションに適用されます：

  • Private Launcher - Android 15のPrivate Space機能に特化した専用ランチャー
  • HomeJump - ホームアプリ（ランチャー）設定画面へ素早くアクセスするためのユーティリティ

これらのアプリケーションは、ユーザーのプライバシーを尊重し、個人情報の収集や保存を行いません。

取得する情報

これらのアプリケーションは、個人情報を収集・追跡・保存することはありません。

分析ツール、広告SDK、サードパーティのトラッキングサービスは一切使用していません。

ローカル保存

両アプリケーションは、AndroidのSharedPreferencesを使用してユーザー設定をデバイスにローカル保存します。これらの設定はユーザーのデバイスにのみ保存され、外部サーバーに送信されることはありません。

Private Launcher

情報の利用

本アプリケーションは、Private Spaceにインストールされているアプリの一覧を取得し、画面に表示するためにのみ使用します。この情報は：

  • すべてデバイス上で処理されます
  • 外部サーバーに送信されることは一切ありません
  • 永続的に保存されません（アプリを開くたびに新規取得）

ローカル保存

  • テーマ設定：ライト、ダーク、システムテーマの設定
  • 言語設定：アプリの表示言語（日本語、英語、スペイン語、システム設定）
  • 自動アンロック設定：アプリ起動時に自動でアンロック要求するかどうか
  • お気に入りアプリ：表示するアプリの選択と表示順序

使用する権限

本アプリケーションは、Private Space専用ランチャーとして機能するために以下の権限を必要とします：

  • ACCESS_HIDDEN_PROFILES：Private Spaceプロファイルの検出とアクセスに必要です。これはPrivate Spaceと連携するアプリ向けに設計されたAndroid 15以降のAPIです。
  • QUERY_ALL_PACKAGES：Private Spaceにインストールされているアプリの一覧を取得するために必要です。これはランチャーアプリに標準的な権限です。

これらの権限は本ランチャーアプリの基本機能に不可欠です。これらの権限を通じて取得した情報は、Private Spaceアプリの表示のみに使用され、外部に送信されることは一切ありません。

デフォルトランチャー設定について

Private Spaceにアクセスするには、本アプリをデフォルトのホームランチャーとして設定する必要があります。これは開発者の選択ではなく、AndroidのPrivate Spaceセキュリティモデルによる要件です。Android設定からいつでもお好みのランチャーに戻すことができます。

HomeJump

情報の利用

本アプリケーションは、ユーザーデータにアクセスしません。Androidのホームアプリ設定画面を開くショートカットを提供するだけです。

ローカル保存

  • メイン画面スキップ設定：メイン画面を表示せず直接ランチャー設定に遷移するかどうか
  • 自動遷移秒数：ランチャー設定画面を自動で開くまでの遅延時間
  • テーマ設定：ライト、ダーク、システムテーマの設定
  • 言語設定：アプリの表示言語（日本語、英語、スペイン語、システム設定）

使用する権限

本アプリケーションは特別な権限を必要としません。

使用する唯一のシステム権限はBIND_QUICK_SETTINGS_TILEで、これはクイック設定タイルを提供するアプリに標準的なAndroid権限です。この権限はユーザーデータにアクセスしません。
