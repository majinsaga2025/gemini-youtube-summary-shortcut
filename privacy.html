Privacy Policy

Gemini YouTube Summary Shortcut is a Chrome extension designed to help users summarize YouTube videos using Gemini. This extension respects user privacy and does not collect or store personal information.

Data Collection

This extension does not collect, track, or store any personal data.

Data Usage

When the user explicitly clicks the extension, the URL of the currently active tab may be sent to Gemini in order to generate a summary. This data is used only for the requested operation and is not stored by this extension.

Local Storage

User-defined prompt settings are stored locally using Chrome's storage API. These settings are never transmitted to any external servers.

Third-Party Services

This extension interacts with Gemini only when initiated by the user. No browsing history, analytics, or tracking information is collected or shared.

Permissions

The extension uses the following permissions:

  • tabs: Used only to open Gemini in a new tab when the user clicks the extension.
  • activeTab: Used only to access the currently active tab after explicit user action.
  • storage: Used to save user-defined prompt settings locally.

Changes to This Policy

This privacy policy may be updated if the extension's functionality changes. Any updates will be reflected on this page.

Contact

If you have any questions about this privacy policy, please contact the developer via the Chrome Web Store listing.

プライバシーポリシー

Gemini YouTube Summary Shortcut は、YouTube動画を Gemini で要約するための Chrome 拡張機能です。 本拡張機能は、ユーザーのプライバシーを尊重し、個人情報の収集や保存を行いません。

取得する情報

本拡張機能は、個人情報を収集・追跡・保存することはありません。

情報の利用

ユーザーが拡張機能を明示的に操作した場合に限り、現在表示中のタブのURLが Gemini に送信されます。 この情報は、要約処理のためだけに使用され、本拡張機能側で保存されることはありません。

ローカル保存

ユーザーが設定したプロンプト内容は、Chrome の storage API を使用してローカルに保存されます。 これらの設定が外部サーバーに送信されることはありません。

外部サービスについて

本拡張機能は、ユーザーの操作に基づいて Gemini と連携します。 閲覧履歴、分析データ、トラッキング情報の収集や共有は行いません。

使用する権限について

本拡張機能は、以下の権限を使用します。

  • tabs：ユーザー操作時に Gemini を新しいタブで開くために使用します。
  • activeTab：ユーザー操作に基づき、現在表示中のタブ情報に一時的にアクセスするために使用します。
  • storage：ユーザーが設定したプロンプトをローカルに保存するために使用します。

ポリシーの変更

本拡張機能の機能変更に伴い、本プライバシーポリシーを更新する場合があります。 変更内容は本ページに反映されます。

お問い合わせ

本プライバシーポリシーに関するお問い合わせは、Chrome ウェブストアの掲載ページを通じてご連絡ください。
